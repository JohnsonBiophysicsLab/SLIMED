\doxysubsection{Detailed Description}
This file includes appended functions to linear algebra package that does not change when switching between LA packages. This mostly includes alias for functions and nested functions. 

\begin{DoxyAuthor}{Author}
Y Ying (\href{mailto:yying7@jh.edu}{\texttt{ yying7@jh.\+edu}}) 
\end{DoxyAuthor}
Note that this section is frequently used in energy and force calculation so improvement of efficiency is highly recommended.

Note that all the functions are in the format of func($\ast$args, result) to avoid allocating new memory to optimize performance.

The error checks are removed in this section for optimized performance by reducing the number of if statements in frequently referenced functions. Therefore, it is unrecommended to use functions in this file in newly implemented section other than energy and force calculation.
