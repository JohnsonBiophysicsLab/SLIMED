{\footnotesize\ttfamily bool Optimization\+Algorithm\+::disable\+\_\+\+NCG\+\_\+if\+\_\+stuck\+\_\+consecutively (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Disables the use of NCG if it has been stuck consecutively over a threshold. 

If the NCG optimizer has been stuck for too many consecutive iterations (as determined by the {\ttfamily is\+NCGstuck} flag), this function will increment a counter {\ttfamily n\+Consecutive\+Ncg\+Stuck}. If that counter exceeds a threshold of n\+Consecutive\+Ncg\+Stuck\+Threshold, the {\ttfamily using\+NCG} flag will be set to false, indicating that NCG should not be used. Otherwise, {\ttfamily using\+NCG} remains true.

If the NCG optimizer is not currently stuck, the {\ttfamily n\+Consecutive\+Ncg\+Stuck} counter is reset to 0 and {\ttfamily using\+NCG} is set to true.

\begin{DoxyReturn}{Returns}
True if NCG should be used, false otherwise.
