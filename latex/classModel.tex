Determines whether the optimization algorithm should continue running. 

The function returns true if the model has not yet been optimized and the maximum number of iterations has not been reached. Otherwise, the function returns false and the optimization algorithm stops.

{\footnotesize\ttfamily void Model\+::update\+\_\+ncg\+\_\+direction (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Updates the direction for nonlinear conjugate gradient (NCG) optimization. 

This function calculates the dot product of the previous and current forces to determine the NCG factor. It then updates the NCG direction either as a combination of the current force and the previous NCG direction, or just the current force depending on whether NCG optimization is being used.
