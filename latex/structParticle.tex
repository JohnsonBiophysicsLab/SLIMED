\doxysubsection{Detailed Description}
Represents a single diffusing particle in a simulation. 

A \mbox{\hyperlink{structParticle}{Particle}} object contains information about the particle\textquotesingle{}s position, velocity, and diffusion constant. The position is represented by a 3x1 matrix of coordinates, while the velocity is represented by another 3x1 matrix of velocities in the x, y, and z directions.
