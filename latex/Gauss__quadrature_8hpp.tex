Calculate the VMU (barycentric coordinates) and weight for a Gaussian quadrature point. 

This function calculates a matrix (N, 3) of VMUs based on the specified number of quadrature points, storing the result in the provided matrix {\ttfamily vmu}. It also calculates a matrix (N, 1) of weights for each point, storing the result in the provided matrix {\ttfamily weight}.

Note that the {\ttfamily vmu} and {\ttfamily weight} matrices will be overwritten and should be initialized before calling this function.

Calculate the shape function and its derivatives at a given barycentric coordinate. 

This function calculates the shape function and its derivatives at a given barycentric coordinate {\ttfamily vwu}, storing the result in the provided matrix {\ttfamily shapefunction}. The shape function and its derivatives are stored in rows 0 to 6 of the matrix as follows\+:

shapefunction(0,\+:), shape functions; shapefunction(1,\+:), differential to v; shapefunction(2,\+:), differential to w; shapefunction(3,\+:), double differential to v; shapefunction(4,\+:), double differential to w; shapefunction(5,\+:), differential to v and w; shapefunction(6,\+:), differential to w and v;
